# react-p2p

Helps manage state between peers in a P2P connections setup with multiple peers.

Includes the following:
- Signalling Server
- SyncProvider
- useSyncedState
- useRoom

## Install

- Bun:
```bash
bun add react-p2p
```
- npm:
```bash
npm install react-p2p
```

## Build

```bash
bun run build
# or
npm run build
```

## Publish

```bash
npm publish
# for scoped public packages
# npm publish --access public
```
